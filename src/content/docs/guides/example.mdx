The einar-cli allows you to generate custom handlers with the necessary files and imports to handle your request. After generating the handler, you must manually replace the placeholder ```/insert-your-custom-pattern-here``` with the actual path you intend to use in your application.

## üë®‚Äçüíª Generate a New Custom Handler

Inside your project directory, run the following command to create a new custom handler:
<Tabs>
  <TabItem label="GET">
```sh
einar generate get-controller get-customer
```

Here's an example of how the generated code might look:
```sh
func init() {
	ioc.Registry(getCustomer, serverwrapper.NewEchoWrapper)
}
func getCustomer(e serverwrapper.EchoWrapper) {
	e.GET("/insert-your-custom-pattern-here", func(c echo.Context) error {
		return c.JSON(http.StatusOK, map[string]string{
			"message": "Unimplemented",
		})
	})
}
```
The file ```get_customer.go``` will be created in the following directory structure:
```
/app
  /adapter
    /in
      /api
        - get_customer.go  
```
  </TabItem>
